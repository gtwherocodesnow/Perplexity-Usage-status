Installation
3 steps, under a minute
01
Download the extension
Click Download Extension above. You'll get a .zip file ‚Äî save it somewhere easy to find like your Desktop or Downloads folder. Then unzip it (double-click on Mac, right-click ‚Üí Extract All on Windows).

You should see a folder called perplexity-usage-ext containing manifest.json, content.js, popup.html, popup.js, and an icons folder.
02
Load it into Chrome
Open a new tab and go to chrome://extensions. In the top-right corner, toggle on Developer Mode. Then click Load unpacked and select the perplexity-usage-ext folder (not the zip, the folder inside it).
Developer mode is required to install extensions from outside the Chrome Web Store. It's safe ‚Äî you can turn it back off after installing if you prefer.
03
Open Perplexity and click the button
Go to perplexity.ai while logged in. You'll see a small bar chart button in the bottom-right corner of the page. Click it ‚Äî your usage panel opens instantly.
Works in the Perplexity web app (PWA) too ‚Äî the button appears directly in the interface so you don't need a browser toolbar.
